# LotScout Data Sources

## Fulton County (30310, 30312, 30314)
Neighborhoods: West End, Pittsburgh, Adair Park, Capitol View

- **Parcel Search**: https://qpublic.schneidercorp.com/Application.aspx?App=FultonCountyGA&Layer=Parcels&PageType=Search
- **GIS/Open Data (downloadable parcels)**: https://gisdata.fultoncountyga.gov/datasets/tax-parcels
- **Tax Assessor**: https://fultonassessor.org/
- **Tax Bill Lookup**: https://www.fultoncountytaxes.org/property-taxes.aspx

## DeKalb County (30079, 30032, 30033)
Neighborhoods: Scottdale, Decatur corridor

- **Parcel Search**: https://qpublic.schneidercorp.com/Application.aspx?App=DekalbCountyGA&Layer=Parcels&PageType=Search
- **Property Appraisal/Map**: https://propertyappraisal.dekalbcountyga.gov/maps/mapadv.aspx
- **GIS Parcel Viewer**: https://www.arcgis.com/apps/webappviewer/index.html?id=f241af753f414cdfa31c1fdef0924584
- **Tax Search**: https://publicaccess.dekalbtax.org/Search/Properties.aspx
- **Deed/Land Records**: https://lrs.dekalbrecorder.com/
